# Karship VS Code Extension

Single extension package for Karship:

- language id: `karship`
- file extensions: `.ksharp`, `.kpp`, `.k`
- grammar: `syntaxes/karship.tmLanguage.json`
- icon theme: `icons/theme.json`

To install locally, copy this folder to:

`%USERPROFILE%\.vscode\extensions\karship.karship-vscode-0.1.0`
